En esta evaluación integrarás todos los conocimientos adquiridos para diseñar, implementar y automatizar un entorno productivo en la nube utilizando AWS y GitHub Actions. El objetivo es demostrar tu dominio en orquestación de contenedores y despliegue automatizado de software.

Deberás realizar las siguientes tareas:

1. Orquestación de contenedores:
* Configurar un clúster en AWS (EKS o ECS).
* Desplegar una aplicación contenedorizada asegurando su escalabilidad y alta disponibilidad.
2. Automatización del pipeline:
* Crear un pipeline CI/CD con GitHub Actions que compile, publique y despliegue la aplicación automáticamente en el clúster configurado.
* Verificar que el flujo de despliegue funcione correctamente (desde el commit hasta la ejecución del servicio).
3. Evaluación y mejora:
* Analizar el desempeño del pipeline e identificar oportunidades de optimización (tiempos, pasos redundantes, errores de ejecución).
4. Presentación final:
* Presentacion con defensa tecnica  (10–15 minutos) donde muestres el pipeline en ejecución, la orquestación de contenedores y reflexiones sobre el proceso de mejora continua.

Esta evaluación te permitirá demostrar competencias integrales en DevOps, comprendiendo cómo automatizar, desplegar y optimizar software productivo en entornos reales de nube, aplicando buenas prácticas profesionales.
